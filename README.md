# Task Manager Backend

## Deskripsi
Aplikasi backend sederhana untuk mengelola data tugas (Task Manager) menggunakan Node.js, Express, dan PostgreSQL.

## Fitur
- Menampilkan semua data tugas
- Menampilkan tugas berdasarkan ID
- Menambah tugas baru
- Mengupdate tugas
- Menghapus tugas

## Endpoint API
- GET /tasks
- GET /tasks/:id
- POST /tasks
- PUT /tasks/:id
- DELETE /tasks/:id

## Teknologi
- Node.js
- Express.js
- PostgreSQL
- Nodemon

## Cara Menjalankan
1. Install dependencies:
   npm install

2. Jalankan server:
   npm start

3. Akses:
   http://localhost:3000
